# @madoc/core

> Markdown Documentation Framework

## Installation

Madoc is built with [Svelte], so you'll need it as a dev dependency.

``` bash
npm install --save-dev @madoc/core svelte
```

[Svelte]: https://svelte.dev/ "Svelte Website"
